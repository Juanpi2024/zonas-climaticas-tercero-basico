# Misión Tierra: Exploradores del Planeta 🌍

Esta es una aplicación didáctica diseñada para estudiantes de **3ro Básico**, basada en la **Unidad 1 de Historia (Mes de Abril)**.

## Características
- **Misiones Interactivas**: Puntos Cardinales, Líneas de Referencia, Continentes y Océanos, Zonas Climáticas.
- **Diseño Premium**: Estética espacial con Glassmorphism y animaciones fluidas.
- **Evaluación Gamificada**: Un "Examen Final" para poner a prueba los conocimientos adquiridos.
- **Sistema de Puntuación**: Incentiva el aprendizaje mediante puntos y feedback positivo.

## Cómo Ejecutar
Simplemente abre el archivo `index.html` en cualquier navegador moderno.
